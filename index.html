Peta Konsep

Dua ide besar yang akan kamu kuasai

Barisan Aritmetika

Mencari nilai pada satu urutan tertentu — misalnya kursi di baris ke-12.

Un = a + (n−1)b

Deret Aritmetika

Mencari jumlah total dari beberapa suku sekaligus — misalnya total seluruh kursi.

Sn = n/2 (a + Un)
Bagian II

Barisan Aritmetika

Barisan aritmetika adalah susunan bilangan dengan pola penambahan atau pengurangan yang selalu tetap. Selisih tetap ini disebut Beda (b), dicari dengan b = Un − Un₋1.

Rumus Suku ke-n
Un = a + (n − 1)b
a = suku pertama · b = beda · n = urutan suku · Un = nilai suku ke-n
Un

Kalkulator Barisan Aritmetika

Masukkan suku pertama, beda, dan urutan suku yang ingin dicari.

    a = 2, b = 5 − 2 = 3
    U10 = 2 + (10 − 1)(3)
    U10 = 2 + 27 = 29
    Suku ke-10 dari barisan tersebut adalah 29.
    a = 50, b = 45 − 50 = −5
    U15 = 50 + (15 − 1)(−5)
    U15 = 50 − 70 = −20
    Suku ke-15 dari barisan tersebut adalah −20.
    U7 − U3 = 4b → 23 − 11 = 4b → b = 3
    U3 = a + 2b → 11 = a + 6 → a = 5
    Un = 5 + (n − 1)(3) = 3n + 2
    Rumus suku ke-n barisan tersebut adalah Un = 3n + 2.
    U9 − U4 = 5b → 38 − 18 = 5b → b = 4
    U4 = a + 3b → 18 = a + 12 → a = 6
    U20 = 6 + 19(4) = 6 + 76 = 82
    Suku ke-20 dari barisan tersebut adalah 82.
    a = 4, b = 5, Un = 124
    124 = 4 + (n − 1)(5)
    120 = (n − 1)(5) → n − 1 = 24 → n = 25
    Banyaknya suku pada barisan tersebut adalah 25 suku.
    a = 5.000, b = 1.000, dicari U15
    U15 = 5.000 + (14)(1.000)
    U15 = 5.000 + 14.000 = 19.000
    Pada hari ke-15, Andi menabung sebesar Rp19.000.
    a = 20, b = 4, dicari U12
    U12 = 20 + (11)(4)
    U12 = 20 + 44 = 64
    Terdapat 64 kursi pada baris ke-12.
    a = 1.200.000, b = 150.000, dicari U8
    U8 = 1.200.000 + (7)(150.000)
    U8 = 1.200.000 + 1.050.000 = 2.250.000
    Gaji karyawan tersebut pada bulan ke-8 adalah Rp2.250.000.
    a = 40, b = 25, dicari U10
    U10 = 40 + (9)(25)
    U10 = 40 + 225 = 265
    Tinggi bambu pada minggu ke-10 adalah 265 cm.
    a = 18.000.000, b = −1.500.000, dicari U6
    U6 = 18.000.000 + (5)(−1.500.000)
    U6 = 18.000.000 − 7.500.000 = 10.500.000
    Perkiraan nilai jual motor pada tahun ke-6 adalah Rp10.500.000.

    Uji Pemahamanmu — Barisan Aritmetika

    Isi jawabanmu, lalu klik "Cek Jawaban". Kalau masih bingung, buka penjelasannya.

    1.Tentukan suku ke-12 dari barisan aritmetika: 7, 12, 17, 22, ...
    a = 7, b = 5 → U12 = 7 + (11)(5) = 7 + 55 = 62
    2.Barisan aritmetika dengan suku pertama 10 dan beda −3. Tentukan suku ke-9.
    a = 10, b = −3 → U9 = 10 + (8)(−3) = 10 − 24 = −14
    3.Perpustakaan menambah koleksi buku tiap bulan. Bulan pertama 150 buku, bertambah 30 buku tiap bulan. Berapa jumlah buku pada bulan ke-11?
    a = 150, b = 30 → U11 = 150 + (10)(30) = 150 + 300 = 450 buku
    4.Sewa ruko tahun pertama Rp24.000.000, naik Rp2.000.000 tiap tahun. Berapa harga sewa tahun ke-7? (isi dalam rupiah tanpa titik, contoh: 30000000)
    a = 24.000.000, b = 2.000.000 → U7 = 24.000.000 + (6)(2.000.000) = Rp36.000.000
    Bagian III

    Deret Aritmetika

    Jika barisan mencari nilai satu suku, deret mencari jumlah total dari n suku pertama. Jumlah ini disimbolkan Sn.

    Rumus Deret Aritmetika
    Sn = n/2 (2a + (n − 1)b)  atau  Sn = n/2 (a + Un)
    Sn = jumlah n suku pertama
    Sn

    Kalkulator Deret Aritmetika

    Masukkan suku pertama, beda, dan banyak suku yang ingin dijumlahkan.

      a = 3, b = 4, n = 10
      S10 = 10/2 (2(3) + (9)(4))
      S10 = 5(6 + 36) = 5 × 42 = 210
      Jumlah 10 suku pertama deret tersebut adalah 210.
      a = 100, b = −5, n = 20
      S20 = 20/2 (2(100) + (19)(−5))
      S20 = 10(200 − 95) = 10 × 105 = 1.050
      Jumlah 20 suku pertama deret tersebut adalah 1.050.
      S13 = 13/2 (5 + 65)
      S13 = 13/2 × 70 = 13 × 35 = 455
      Jumlah deret tersebut (S13) adalah 455.
      144 = 8/2 (2(4) + 7b) = 4(8 + 7b)
      36 = 8 + 7b → 7b = 28 → b = 4
      U8 = 4 + 7(4) = 32
      Beda deret tersebut adalah 4, dan U8 = 32.
      3, 6, 9, ..., 99 → a = 3, b = 3
      99 = 3 + (n − 1)(3) → n = 33
      S33 = 33/2 (3 + 99) = 33 × 51 = 1.683
      Jumlah semua kelipatan 3 antara 1 dan 100 adalah 1.683.
      a = 20, b = 4, n = 15
      S15 = 15/2 (40 + 56)
      S15 = 15/2 × 96 = 15 × 48 = 720
      Kapasitas total gedung pertunjukan adalah 720 kursi.
      a = 5.000, b = 1.000, n = 20
      S20 = 10(10.000 + 19.000)
      S20 = 10 × 29.000 = 290.000
      Total tabungan Andi selama 20 hari pertama adalah Rp290.000.
      a = 2.000.000, b = −100.000, n = 12
      U12 = 2.000.000 − 1.100.000 = 900.000
      S12 = 6(2.000.000 + 900.000) = 17.400.000
      Total cicilan yang dibayarkan selama 12 bulan adalah Rp17.400.000.
      a = 500, b = 50, n = 12
      U12 = 500 + 550 = 1.050
      S12 = 6(500 + 1.050) = 6 × 1.550 = 9.300
      Total produksi pabrik selama 12 bulan pertama adalah 9.300 unit.
      a = 2, b = 0,5, n = 10
      U10 = 2 + 4,5 = 6,5
      S10 = 5(2 + 6,5) = 5 × 8,5 = 42,5
      Total jarak yang ditempuh selama 10 hari latihan adalah 42,5 km.
      Latihan Mandiri

      Uji Pemahamanmu — Deret Aritmetika

      1.Hitung jumlah 15 suku pertama dari deret: 6 + 10 + 14 + 18 + ...
      a=6, b=4, n=15 → S15 = 15/2(12 + 56) = 15/2 × 68 = 510
      2.Deret aritmetika dengan a = 8, b = 5, n = 10. Tentukan S10.
      S10 = 10/2(2(8) + 9(5)) = 5(16 + 45) = 5 × 61 = 305
      3.Koperasi menjual buku: hari pertama 12, bertambah 3 buku/hari. Total terjual selama 14 hari?
      U14 = 12 + 13(3) = 51; S14 = 14/2(12 + 51) = 7 × 63 = 441 buku
      4.Toko kue: hari pertama 40 donat, bertambah 5 donat/hari, selama 20 hari. Total produksi?
      U20 = 40 + 19(5) = 135; S20 = 20/2(40 + 135) = 10 × 175 = 1.750 donat
      Bagian IV

      Rangkuman & Peta Rumus

      Barisan Aritmetika
      Un = a + (n − 1)b
      mencari nilai satu suku tertentu
      Deret Aritmetika
      Sn = n/2 (2a + (n−1)b) = n/2 (a + Un)
      mencari jumlah total n suku pertama

      Tips Mengingat

      • b positif → barisan naik (contoh: tabungan, gaji, produksi).
      • b negatif → barisan turun (contoh: penyusutan nilai barang, cicilan menurun).
      • Kata kunci "pada urutan/hari/bulan/baris ke-..." biasanya mengarah ke rumus Un.
      • Kata kunci "total/jumlah keseluruhan selama..." biasanya mengarah ke rumus Sn.

Mulai Belajar Aritmetika → Langsung ke Geometri

Coba geser bedanya

Un = 4 + (n−1)·b
3
Peta Konsep

Empat ide besar yang akan kamu kuasai

Bedanya cuma satu: Aritmetika bertambah dengan penjumlahan tetap, Geometri bertambah dengan perkalian tetap.

Barisan Aritmetika

Mencari nilai pada satu urutan tertentu, dengan selisih tetap — misalnya kursi di baris ke-12.

Un = a + (n−1)b

Deret Aritmetika

Mencari jumlah total dari beberapa suku sekaligus — misalnya total seluruh kursi.

Sn = n/2 (a + Un)

Barisan Geometri

Mencari nilai pada satu urutan tertentu, dengan rasio (perkalian) tetap — misalnya jumlah bakteri jam ke-6.

Un = a · rn−1

Deret Geometri

Mencari jumlah total dari beberapa suku sekaligus — misalnya total hadiah berlipat ganda.

Sn = a(rn−1) / (r−1)
Bagian II

Barisan Aritmetika

Barisan aritmetika adalah susunan bilangan dengan pola penambahan atau pengurangan yang selalu tetap. Selisih tetap ini disebut Beda (b), dicari dengan b = Un − Un₋1.

Rumus Suku ke-n
Un = a + (n − 1)b
a = suku pertama · b = beda · n = urutan suku · Un = nilai suku ke-n
Un

Kalkulator Barisan Aritmetika

Masukkan suku pertama, beda, dan urutan suku yang ingin dicari.

    a = 2, b = 5 − 2 = 3
    U10 = 2 + (10 − 1)(3)
    U10 = 2 + 27 = 29
    Suku ke-10 dari barisan tersebut adalah 29.
    a = 50, b = 45 − 50 = −5
    U15 = 50 + (15 − 1)(−5)
    U15 = 50 − 70 = −20
    Suku ke-15 dari barisan tersebut adalah −20.
    U7 − U3 = 4b → 23 − 11 = 4b → b = 3
    U3 = a + 2b → 11 = a + 6 → a = 5
    Un = 5 + (n − 1)(3) = 3n + 2
    Rumus suku ke-n barisan tersebut adalah Un = 3n + 2.
    U9 − U4 = 5b → 38 − 18 = 5b → b = 4
    U4 = a + 3b → 18 = a + 12 → a = 6
    U20 = 6 + 19(4) = 6 + 76 = 82
    Suku ke-20 dari barisan tersebut adalah 82.
    a = 4, b = 5, Un = 124
    124 = 4 + (n − 1)(5)
    120 = (n − 1)(5) → n − 1 = 24 → n = 25
    Banyaknya suku pada barisan tersebut adalah 25 suku.
    a = 5.000, b = 1.000, dicari U15
    U15 = 5.000 + (14)(1.000)
    U15 = 5.000 + 14.000 = 19.000
    Pada hari ke-15, Andi menabung sebesar Rp19.000.
    a = 20, b = 4, dicari U12
    U12 = 20 + (11)(4)
    U12 = 20 + 44 = 64
    Terdapat 64 kursi pada baris ke-12.
    a = 1.200.000, b = 150.000, dicari U8
    U8 = 1.200.000 + (7)(150.000)
    U8 = 1.200.000 + 1.050.000 = 2.250.000
    Gaji karyawan tersebut pada bulan ke-8 adalah Rp2.250.000.
    a = 40, b = 25, dicari U10
    U10 = 40 + (9)(25)
    U10 = 40 + 225 = 265
    Tinggi bambu pada minggu ke-10 adalah 265 cm.
    a = 18.000.000, b = −1.500.000, dicari U6

    a = 24.000.000, b = 2.000.000 → U7 = 24.000.000 + (6)(2.000.000) = Rp36.000.000
    Bagian III

    Deret Aritmetika

    Jika barisan mencari nilai satu suku, deret mencari jumlah total dari n suku pertama. Jumlah ini disimbolkan Sn.

    Rumus Deret Aritmetika
    Sn = n/2 (2a + (n − 1)b)  atau  Sn = n/2 (a + Un)
    Sn = jumlah n suku pertama
    Sn

    Kalkulator Deret Aritmetika

    Masukkan suku pertama, beda, dan banyak suku yang ingin dijumlahkan.

      a = 3, b = 4, n = 10
      S10 = 10/2 (2(3) + (9)(4))
      S10 = 5(6 + 36) = 5 × 42 = 210
      Jumlah 10 suku pertama deret tersebut adalah 210.
      a = 100, b = −5, n = 20
      S20 = 20/2 (2(100) + (19)(−5))
      S20 = 10(200 − 95) = 10 × 105 = 1.050
      Jumlah 20 suku pertama deret tersebut adalah 1.050.
      S13 = 13/2 (5 + 65)
      S13 = 13/2 × 70 = 13 × 35 = 455
      Jumlah deret tersebut (S13) adalah 455.
      144 = 8/2 (2(4) + 7b) = 4(8 + 7b)
      36 = 8 + 7b → 7b = 28 → b = 4
      U8 = 4 + 7(4) = 32
      Beda deret tersebut adalah 4, dan U8 = 32.
      3, 6, 9, ..., 99 → a = 3, b = 3
      99 = 3 + (n − 1)(3) → n = 33
      S33 = 33/2 (3 + 99) = 33 × 51 = 1.683
      Jumlah semua kelipatan 3 antara 1 dan 100 adalah 1.683.
      a = 20, b = 4, n = 15
      S15 = 15/2 (40 + 56)
      S15 = 15/2 × 96 = 15 × 48 = 720
      Kapasitas total gedung pertunjukan adalah 720 kursi.
      a = 5.000, b = 1.000, n = 20
      S20 = 10(10.000 + 19.000)
      S20 = 10 × 29.000 = 290.000
      Total tabungan Andi selama 20 hari pertama adalah Rp290.000.
      a = 2.000.000, b = −100.000, n = 12
      U12 = 2.000.000 − 1.100.000 = 900.000
      S12 = 6(2.000.000 + 900.000) = 17.400.000
      Total cicilan yang dibayarkan selama 12 bulan adalah Rp17.400.000.

      a = 2, b = 0,5, n = 10

      Bagian IV

      Barisan Geometri

      Barisan geometri adalah susunan bilangan dengan pola perkalian tetap antar suku. Perkalian tetap ini disebut Rasio (r), dicari dengan r = Un ÷ Un₋1.

      Rumus Suku ke-n
      Un = a · rn−1
      a = suku pertama · r = rasio · n = urutan suku · Un = nilai suku ke-n
      Un

      Kalkulator Barisan Geometri

      Masukkan suku pertama, rasio, dan urutan suku yang ingin dicari.

        a = 3, r = 6 ÷ 3 = 2
        U7 = 3 × 2⁶
        U7 = 3 × 64 = 192
        Suku ke-7 dari barisan tersebut adalah 192.
        a = 243, r = 81 ÷ 243 = 1/3
        U6 = 243 × (1/3)⁵
        U6 = 243 ÷ 243 = 1
        Suku ke-6 dari barisan tersebut adalah 1.
        U5 ÷ U2 = r³ → 48 ÷ 6 = r³ → r = 2
        U2 = a·r → 6 = a(2) → a = 3
        Un = 3 × 2ⁿ⁻¹
        Rumus suku ke-n barisan tersebut adalah Un = 3 × 2ⁿ⁻¹.
        U6 ÷ U3 = r³ → 160 ÷ 20 = r³ → r = 2
        U3 = a·r² → 20 = a(4) → a = 5
        U8 = 5 × 2⁷ = 5 × 128 = 640
        Suku ke-8 dari barisan tersebut adalah 640.
        a = 2, r = 3, Un = 486
        486 = 2 × 3ⁿ⁻¹ → 3ⁿ⁻¹ = 243 = 3⁵
        n − 1 = 5 → n = 6
        Banyaknya suku pada barisan tersebut adalah 6 suku.
        a = 5, r = 2, dicari U7 (jam ke-6 = suku ke-7)
        U7 = 5 × 2⁶
        U7 = 5 × 64 = 320
        Jumlah bakteri pada jam ke-6 adalah 320 bakteri.
        a = 8, r = 1/2, dicari U5
        U5 = 8 × (1/2)⁴
        U5 = 8 ÷ 16 = 0,5
        Tinggi pantulan ke-5 adalah 0,5 meter.
        a = 1, r = 3, dicari U5
        U5 = 1 × 3⁴
        U5 = 81
        Penerima baru pada jam ke-5 sebanyak 81 orang.
        a = 2.000.000, r = 1,1, dicari U4
        U4 = 2.000.000 × 1,1³
        U4 = 2.000.000 × 1,331 = 2.662.000
        Nilai investasi pada tahun ke-4 adalah Rp2.662.000.
        a = 4, r = 2, dicari U6
        U6 = 4 × 2⁵
        U6 = 4 × 32 = 128
        Populasi kelinci pada bulan ke-6 adalah 128 ekor.

        Uji Pemahamanmu — Barisan Geometri

        Isi jawabanmu, lalu klik "Cek Jawaban".

        1.Tentukan suku ke-5 dari barisan geometri: 4, 8, 16, ...
        a = 4, r = 2 → U5 = 4 × 2⁴ = 4 × 16 = 64
        2.Barisan geometri dengan suku pertama 3 dan rasio 2. Tentukan suku ke-8.
        a = 3, r = 2 → U8 = 3 × 2⁷ = 3 × 128 = 384
        3.Virus komputer mula-mula menginfeksi 2 komputer, tiap jam menjadi 3× lipat. Berapa komputer terinfeksi pada jam ke-5 (U5)?
        a = 2, r = 3 → U5 = 2 × 3⁴ = 2 × 81 = 162
        4.Mesin fotokopi memperkecil gambar jadi setengah tiap kali. Ukuran awal 80 cm. Berapa ukuran setelah 4 kali pengecilan (U5)?
        a = 80, r = 1/2 → U5 = 80 × (1/2)⁴ = 80 ÷ 16 = 5 cm
        Bagian V

        Deret Geometri

        Sama seperti deret aritmetika, deret geometri mencari jumlah total dari n suku pertama sebuah barisan geometri.

        Rumus Deret Geometri
        Sn = a(rn − 1) / (r − 1)  atau  Sn = a(1 − rn) / (1 − r)
        gunakan bentuk kedua bila 0 < r < 1, agar hasil bagi tetap positif
        Sn

        Kalkulator Deret Geometri

        Masukkan suku pertama, rasio, dan banyak suku yang ingin dijumlahkan.

          a = 2, r = 3, n = 6
          S6 = 2(3⁶ − 1) / (3 − 1)
          S6 = 2(728) / 2 = 728
          Jumlah 6 suku pertama deret tersebut adalah 728.
          a = 96, r = 1/2, n = 5
          S5 = 96(1 − (1/2)⁵) / (1 − 1/2)
          S5 = 96(31/32) × 2 = 186
          Jumlah 5 suku pertama deret tersebut adalah 186.
          S4 = 5(3⁴ − 1) / (3 − 1)
          S4 = 5(80) / 2
          S4 = 400 / 2 = 200
          Jumlah 4 suku pertama deret tersebut (S4) adalah 200.
          26 = 2(1 + r + r²) → 1 + r + r² = 13
          r² + r − 12 = 0 → (r+4)(r−3) = 0 → r = 3
          U3 = a·r² = 2 × 9 = 18
          Rasio deret tersebut adalah 3, dan U3 = 18.
          a = 1, r = 2, Un = 256 → 2ⁿ⁻¹ = 256 = 2⁸ → n = 9
          S9 = 1(2⁹ − 1) / (2 − 1)
          S9 = 511
          Jumlah deret tersebut adalah 511.
          a = 8, r = 1/2, n = 4
          S4 = 8(1 − (1/2)⁴) / (1 − 1/2)
          S4 = 8(15/16) × 2 = 15
          Total tinggi 4 pantulan pertama adalah 15 meter.
          a = 1, r = 3, n = 5
          S5 = 1(3⁵ − 1) / (3 − 1)
          S5 = 242 / 2 = 121
          Total orang yang telah menerima pesan adalah 121 orang.
          a = 100.000, r = 2, n = 6
          S6 = 100.000(2⁶ − 1) / (2 − 1)
          S6 = 100.000 × 63 = 6.300.000
          Total hadiah yang dibagikan selama 6 babak adalah Rp6.300.000.
          a = 1.000, r = 2, n = 8
          S8 = 1.000(2⁸ − 1) / (2 − 1)
          S8 = 1.000 × 255 = 255.000
          Total tabungan selama 8 hari tersebut adalah Rp255.000.
          a = 512, r = 1/2, n = 5
          S5 = 512(1 − (1/2)⁵) / (1 − 1/2)
          S5 = 512(31/32) × 2 = 992
          Total udara yang dikeluarkan setelah 5 tarikan adalah 992 ml.

          Uji Pemahamanmu — Deret Geometri

          1.Hitung jumlah 5 suku pertama dari deret geometri: 3 + 6 + 12 + 24 + ...
          a=3, r=2, n=5 → S5 = 3(2⁵−1)/(2−1) = 3 × 31 = 93
          2.Deret geometri dengan a = 4, r = 3, n = 4. Tentukan S4.
          S4 = 4(3⁴−1)/(3−1) = 4(80)/2 = 160
          3.Turnamen esport: hadiah babak 1 Rp50.000, berlipat 2× tiap babak, 5 babak. Total hadiah?
          S5 = 50.000(2⁵−1)/(2−1) = 50.000 × 31 = Rp1.550.000
          4.Bola: tinggi pantulan pertama 10 m, rasio 2/5. Total tinggi 3 pantulan pertama? (boleh desimal, contoh: 15.6)
          S3 = 10(1 − (2/5)³)/(1 − 2/5) = 10(0,936)/(0,6) = 15,6 m
          Bagian VI

          Rangkuman & Peta Rumus

          Barisan Aritmetika
          Un = a + (n − 1)b
          mencari nilai satu suku tertentu
          Deret Aritmetika
          Sn = n/2 (2a + (n−1)b) = n/2 (a + Un)
          mencari jumlah total n suku pertama
          Barisan Geometri
          Un = a · rn−1
          mencari nilai satu suku tertentu
          Deret Geometri
          Sn = a(rn−1) / (r−1)
          mencari jumlah total n suku pertama

          Tips Mengingat

          • Aritmetika → antar suku selisihnya tetap dijumlah/dikurang (beda, b). Geometri → antar suku tetap dikali/dibagi (rasio, r).
          • b atau r positif > 1 → barisan naik (tabungan, gaji, bakteri, investasi). 0 < r < 1 → barisan menurun mendekati nol (pantulan bola, penyusutan).
          • Kata kunci "pada urutan/hari/bulan/baris/jam ke-..." biasanya mengarah ke rumus Un.
          • Kata kunci "total/jumlah keseluruhan selama..." biasanya mengarah ke rumus Sn.
          • Cek dulu polanya: selisih antar suku tetap → Aritmetika. Hasil bagi antar suku tetap → Geometri.
